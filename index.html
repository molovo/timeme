TimeMe

Simple, non-invasive JavaScript time sanitisation.

The time is validated on blur, and converted into an 'Nh Nm' format for easy reading, from an array of accepted inputs. Optionally, the time can then be converted to an integer minute value on form submit, to allow it to be stored as such in the database and used in calculations.

Try it out now.

If the time is valid, the field is updated. If not, a class of error is applied to the input.

To implement, call the script on your page, and apply a class of timeMe to each input element you want validated. To convert the time to minutes on form submit, add a class of timeMe-form to the parent form element. Simples 


<form action="" class="timeMe-form">
  <input type="text" class="timeMe" />
  <script src="js/timeme.min.js"></script>
</form>

Accepted input formats:

  • 12  // Default is minutes
  • 12m
  • 12h
  • 12u  // u is a 'unit of time'. Default is 6m
  • 12h8
  • 12h8m
  • 12h 8
  • 12h 8m
